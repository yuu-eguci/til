Pipenv Note 
===

## Commands

```bash
# Get pipenv
pip install pipenv
```

```bash
# Create virtual env.

# If you don't have neither Pipfile nor Pipfile.lock
# You have to have the python version in your PC.
pipenv install --python 3.7

# If you have Pipfile.lock
pipenv install
```

```bash
# Install pip modules.
pipenv install MODULE1 MODULE2 ...
```
